# Hero background image

Drop your featured "completed job" photo here, named `featured-job.jpg`
(or update the path in `src/components/home/Hero.jsx`).

Recommended: a wide landscape/hardscaping shot, at least 1920x1080,
compressed for web. The hero overlays a dark gradient on top of it, so
photos with good contrast in the lower-left third (where the headline
sits) work best.

Until you add a photo, the hero shows a soft brand-green placeholder
pattern instead of a broken image.
